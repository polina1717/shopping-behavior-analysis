# shopping-behavior-analysis
Анализ поведения клиентов на основе датасета покупок
Анализ данных о покупательском поведении

Описание проекта
Проект представляет собой комплексный анализ набора данных о покупательском поведении, содержащего информацию о 1000+ транзакциях.

Структура данных
| Customer ID | int | Уникальный идентификатор клиента |
| Age | int | Возраст клиента |
| Gender | str | Пол клиента |
| Item Purchased | str | Название купленного товара |
| Category | str | Категория товара |
| Purchase Amount (USD) | float | Сумма покупки в долларах |
| Location | str | Штат покупки |
| Size | str | Размер товара |
| Color | str | Цвет товара |
| Season | str | Сезон покупки |
| Review Rating | float | Рейтинг отзыва (1-5) |
| Subscription Status | str | Статус подписки (Yes/No) |
| Shipping Type | str | Тип доставки |
| Discount Applied | str | Применена ли скидка |
| Promo Code Used | str | Использован ли промокод |
| Previous Purchases | int | Количество предыдущих покупок |
| Payment Method | str | Метод оплаты |
| Frequency of Purchases | str | Частота покупок |


1. Подготовка данных
- Проверка и обработка пропущенных значений
- Поиск и удаление дубликатов
- Анализ и обработка выбросов (метод IQR)
- Преобразование типов данных
2. Статистический анализ
- Описательная статистика (среднее, медиана, стандартное отклонение)
- Корреляционный анализ между числовыми переменными
- Частотный анализ категориальных переменных

3. Визуализация
- Гистограммы распределений
- Столбчатые диаграммы
- Линейные графики трендов
- Boxplot диаграммы
- Тепловые карты корреляций
- Круговые диаграммы

4. Группировки и агрегации
- Анализ по возрастным группам
- Статистика по категориям товаров
- Распределение по сезонам
- Анализ методов оплаты
- Сводные таблицы

- Средняя сумма покупки: $61.57
- Самый активный сезон: Winter
- Самый дорогой сезон: Spring (средняя покупка $63.38)

Методы оплаты
- Самый популярный метод: Credit Card (36.5%)
- Наименее популярный: Bank Transfer (11.6%)
